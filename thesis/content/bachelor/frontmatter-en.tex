\thesisabstract{%
Edge devices used in smart environments must respond quickly while operating with limited memory and a clearly defined security boundary. This bachelor thesis designs and implements a modular prototype that separates sensing, local inference, security policy, and presentation, and evaluates the components under a reproducible protocol. In the illustrative evaluation, the optimised configuration reaches 91.0\% accuracy with 29~ms latency and 241~MB memory, compared with 72.0\%, 84~ms, and 410~MB for the baseline; these values are placeholders to replace with real results. The main contribution is an evidence-linked design and evaluation workflow that makes performance, resource, and security trade-offs explicit.
}
